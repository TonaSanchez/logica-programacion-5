# logica-programacion-5
Diccionario Español - Inglés

Este proyecto consiste en un programa desarrollado en Java que funciona como un mini diccionario Español-Inglés.

Descripción

El programa:

Crea un diccionario con palabras en español y su traducción al inglés.
Utiliza un HashMap para almacenar las palabras.
Selecciona 5 palabras aleatorias.
Solicita al usuario escribir la traducción correcta.
Verifica las respuestas.
Muestra la cantidad de respuestas correctas e incorrectas al finalizar.
Tecnologías utilizadas
Java
IntelliJ IDEA / VS Code / Eclipse
Colecciones (HashMap, ArrayList)
Scanner
Random
Conceptos aplicados
Variables
Condicionales (if)
Ciclos (for)
Entrada de datos con Scanner
Uso de HashMap
Generación de números aleatorios
